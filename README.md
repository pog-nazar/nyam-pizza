# Лендинг піцерії "Ням-Ням"

Готовий односторінковий сайт для піцерії. Мінімалістичний дизайн, мікроанімації, повна адаптивність під мобільні, планшети й десктопи.

---

## Структура проекту

```
nyam-nyam/
├── index.html          ← головна сторінка (весь HTML)
├── css/
│   └── styles.css      ← усі стилі (кольори зверху файлу)
├── js/
│   └── script.js       ← уся логіка (CONFIG зверху файлу)
├── images/             ← сюди класти фото піц та логотип
│   └── README.txt
└── README.md           ← цей файл
```

---

## Як відкрити в VS Code

1. Завантажте папку `nyam-nyam` собі на комп'ютер.
2. Відкрийте VS Code.
3. Меню `File → Open Folder...` → виберіть папку `nyam-nyam`.
4. Зліва побачите всі файли проекту.

## Як подивитися сайт у браузері

### Спосіб 1 — найпростіший
Двічі клікніть на `index.html` у файловому менеджері. Сайт відкриється в браузері.

### Спосіб 2 — з автооновленням (рекомендовано)
Встановіть розширення **Live Server** для VS Code:

1. Відкрийте вкладку Extensions (іконка квадратиків зліва, або `Ctrl+Shift+X`).
2. У пошуку введіть `Live Server` (автор: Ritwick Dey).
3. Натисніть **Install**.
4. Відкрийте `index.html`, натисніть правою кнопкою → **Open with Live Server**.
5. Сайт відкриється на `http://127.0.0.1:5500`. Будь-яка зміна в коді одразу оновить сторінку.

## Що і де змінювати

### 1. Контактні дані, телефон, графік (файл `js/script.js`)
Угорі файлу є об'єкт `CONFIG`. Змініть значення там — і вони підставляться по всьому сайту:

```js
const CONFIG = {
  pizzeriaName: "Ням-Ням",        // назва
  phone: "+380501350651",         // телефон (для WhatsApp та дзвінків)
  address: "вул. Перемоги, 62...",// адреса
  schedule: { openHour: 11, closeHour: 22 }, // графік для статусу "відкрито"
  pizzas: [ ... ]                 // меню для замовлень
};
```

### 2. Кольори (файл `css/styles.css`)
Угорі файлу блок `:root`. Змініть один колір — зміниться весь сайт:

```css
:root {
  --primary-color: #FF6B35;  /* основний помаранчевий */
  --bg-dark: #0F172A;        /* темний фон footer/контактів */
  ...
}
```

Кілька варіантів основного кольору:
- `#FF5722` — глибокий помаранч
- `#D32F2F` — томатно-червоний
- `#F9A825` — золотий

### 3. Тексти піц, ціни, назви (файл `index.html`)
Знайдіть секцію `<!-- MENU -->`. Кожна піца — це блок `<div class="pizza-card">`. Міняйте назву, склад і ціну прямо в тексті. Не забудьте оновити `data-price` у кнопці:

```html
<button class="btn-order" data-pizza="Пепероні" data-price="250">Замовити</button>
```

### 4. Фото піц (файл `index.html`)
Зараз замість фото стоїть текст `PIZZA`. Щоб додати реальне фото:

1. Покладіть файл фото в папку `images/` (наприклад `pepperoni.jpg`).
2. У `index.html` знайдіть рядок:
   ```html
   <div class="pizza-image">PIZZA</div>
   ```
3. Замініть на:
   ```html
   <img src="images/pepperoni.jpg" class="pizza-image" alt="Пепероні">
   ```

### 5. Логотип (файл `index.html`)
Знайдіть у шапці:
```html
<div class="logo-img">Y</div>
```
Замініть на:
```html
<img src="images/logo.png" class="logo-img" alt="Ням-Ням">
```

### 6. Фон банера (файл `css/styles.css`)
Знайдіть `.hero` і замініть градієнт на фото:
```css
.hero {
  background: linear-gradient(rgba(0,0,0,0.4), rgba(0,0,0,0.4)), url('../images/hero.jpg') center/cover;
}
```

### 7. Соцмережі (файл `index.html`)
Знайдіть посилання `<a href="#" class="social-link">` (та `social-icon` у footer) і замініть `#` на ваші посилання, наприклад `https://instagram.com/ваш_профіль`.

## Що вже працює

- Кнопки "Замовити" і форма відкривають WhatsApp із готовим текстом замовлення.
- Статус "Відкрито / Закрито" рахується автоматично за поточним часом.
- Рік у копірайті оновлюється сам.
- Мобільне бургер-меню.
- Header ховається при скролі вниз і з'являється при скролі вгору.
- Картки з'являються з анімацією під час скролу.

## Як завантажити сайт в інтернет (безкоштовно)

### Netlify (найшвидше)
1. Зайдіть на netlify.com, зареєструйтесь.
2. Перетягніть папку `nyam-nyam` у вікно "deploy".
3. Отримаєте посилання виду `ваша-назва.netlify.app`.

### GitHub Pages
1. Створіть репозиторій на github.com, завантажте файли.
2. Settings → Pages → виберіть гілку → Save.
3. Сайт буде на `username.github.io/nyam-nyam`.
